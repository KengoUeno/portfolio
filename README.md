# ◯◯高校野球部トレーニング記録
# アプリケーション概要<br>
高校部活性向けのトレーニング記録サイトです。 <br>
トレーニングのデータ（挙上回数、挙上重量）から最大重量を算出、投稿し、ユーザー間での順位を確認することができます。<br>
